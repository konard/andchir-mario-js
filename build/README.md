# Иконки приложения

Эта папка должна содержать иконки для сборки Electron приложения.

## Требуемые файлы

- `icon.ico` - иконка для Windows (256x256 пикселей в формате ICO)
- `icon.icns` - иконка для macOS (1024x1024 пикселей в формате ICNS)
- `icon.png` - иконка для Linux (512x512 пикселей в формате PNG)

## Создание иконок

Подробные инструкции по созданию иконок см. в файле [BUILD.md](../BUILD.md#настройка-иконок) в корне проекта.

## Примечание

Если иконки отсутствуют, electron-builder будет использовать иконку по умолчанию.
Для профессионального вида приложения рекомендуется создать собственные иконки.
